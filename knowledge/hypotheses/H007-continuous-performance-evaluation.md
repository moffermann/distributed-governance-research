# H007 — Continuous Performance Evaluation

## Hypothesis

Distributed resource allocation should rely not only on project promises, but on historical, verifiable performance.

## Rationale

If citizens or contributors allocate resources directly, they need fast access to past outcomes. Without performance visibility, allocation may be driven by rhetoric, emotional appeal, or incomplete information.

## Mechanism

Each institution or project should have a simple evaluation dashboard showing:

- previous commitments;
- stated goals;
- budget requested and executed;
- KPI completion;
- beneficiary evaluations;
- complaints;
- independent audits;
- evidence of execution;
- comparisons with similar institutions or projects.

## Key distinction

The dashboard should not reduce value to economic return. It should represent broader social value: education quality, care quality, lives improved, suffering reduced, community strengthened, and goals achieved.
