# Open Questions

## Priority questions

1. What formal criteria determine whether a state function should be centralized, distributed, or hybrid?
2. How should legitimacy be produced in distributed justice and legislative systems?
3. How can resource allocation remain distributed while protecting low-visibility but essential services?
4. What information architecture prevents capture by a new information monopoly?
5. What is the minimum central authority required for defense and security?
6. How can institutional performance be measured without reducing social value to narrow economic metrics?
7. What existing literature already covers parts of this theory?
8. What would be the first real-world experiment or MVP?
9. How can a national, regional, communal, or thematic roadmap be constructed through distributed, votable planning areas without recreating centralized planning monopoly or popularity-driven capture?
