# Zenodo supersession notice (for older versions' metadata)

Prepended to the **top of the Description** of each superseded Zenodo version by
`scripts/zenodo-supersede.mjs`. Editing a published record's metadata does **not** mint a
new DOI, so this is safe on any older version. Cite the **concept DOI**
`10.5281/zenodo.21193846` — it always resolves to the latest version (currently v1.14,
`10.5281/zenodo.21311851`).

The script embeds the exact text below (kept here for human reference).

---

## English

> ⚠️ **SUPERSEDED — please see the latest version.** This version reported a calibrated
> compound value-per-budget multiplier that is now **retired as a calibrated effect**. The
> project's simulation contrasts are directional and conditional on a stylized
> comparative-institutions model — not calibrated real-world effects — and it rests on the
> architecture and the qualitative credit-versus-coverage mechanism, not a multiplier.
> Please read the latest version and cite the concept DOI **10.5281/zenodo.21193846** (it
> always resolves to the latest). Controlling specification:
> `research/claim-and-estimand-contract.md`.
