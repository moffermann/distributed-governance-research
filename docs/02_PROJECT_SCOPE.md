# Project Scope

## Working title

**Functional Architecture of a Distributed State**

Alternative titles:

- Toward a Functional Architecture for Distributed Governance
- Distributed Trust and the Functional Redesign of State Institutions
- From Institutional Monopolies to Distributed Governance Networks

## Central question

How should state functions be redesigned when modern technology reduces the cost of coordination, verification, participation, and distributed decision-making?

## Initial thesis

The state should not be analyzed as a single block. Its functions have different natures. Some functions may still require centralization, especially those close to the legitimate use of force. Other functions, especially those related to information processing, deliberation, resource allocation, and institutional accountability, may benefit from distribution.

## Non-goals

This project is not currently trying to:

- advocate chaos or absence of rules;
- eliminate all institutions;
- prove that every state function can be distributed;
- design a final constitution;
- publish a finished political manifesto.

## Current aim

Build a rigorous conceptual framework that separates state functions by their functional nature and identifies where distribution is technically, institutionally, and socially plausible.
